### Module Hero
- Uso: cabecera reutilizable para modulos de gestion como `Personas`, `Reuniones` y `Actas`
- Contenedor: `card-clean module-hero`
- Padding: `1.25rem`
- Fondo: gradiente claro con realce radial sutil
- Estructura: columna izquierda con `dashboard-kicker`, `dashboard-title`, `dashboard-subtitle` y acciones; columna derecha con `module-hero-stats`

### Module Hero Stat
- Clase: `module-hero-stat`
- Radius: `16px`
- Padding: `0.9rem 1rem`
- Borde: `1px solid #e4ebf4`
- Valor principal: `1.35rem`, peso `800`
- Texto de apoyo: `0.84rem`

### Module Panel
- Uso: tarjetas principales de formulario, listado o detalle en modulos de gestion
- Clase base: `card-clean module-panel-card`
- Fondo: `linear-gradient(180deg, #ffffff 0%, #fbfcff 100%)`
- Cabecera: `module-panel-head`
- Titulo: `module-panel-title`
- Subtitulo: `module-panel-subtitle`
- Badge de contexto: `module-panel-badge`

### Sticky Sidebar
- Uso: formularios laterales cuando la pantalla tiene dos columnas
- Clase: `module-sticky-sidebar`
- Offset superior: `88px`
- Comportamiento responsive: se desactiva en `max-width: 768px`

### Module Table
- Uso: tablas principales de gestion y analisis
- Clase: `module-table`
- Encabezado: fondo `#f8fbff`, borde inferior suave, texto en `0.77rem`
- Filas: padding vertical `0.9rem`, hover muy sutil
- Soporte: `sort-header`, `sort-controls`, `module-toolbar`

### Module Pagination
- Uso: navegacion compacta de listados
- Contenedor: `module-pagination`
- Estado: `module-pagination-status`
- Navegacion: `module-pagination-controls`
- Indicador central: `module-pagination-index`
